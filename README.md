# E-Commerce Product Performance Analysis 

Uncovering actionable insights from online sales data to drive strategic growth and enhance customer satisfaction.


## Project Overview

This project focuses on analyzing e-commerce product performance using multiple data analytics tools and technologies.  
The goal is to understand **sales trends, profitability, and customer behavior** through structured data analysis and interactive dashboards.

By integrating **Excel, SQL, Python, and Power BI**, the project delivers meaningful business insights that support data-driven decision-making.


##  Objectives

- Analyze overall sales and profit performance
- Identify top-performing and underperforming products
- Understand customer purchasing behavior
- Track trends across time, categories, and customer segments
- Build interactive dashboards for stakeholders

### Tools & Technologies Used

### Excel
- Data cleaning and validation  
- Pivot tables and charts  
- Excel dashboard for quick insights  

###  SQL
- Data extraction and transformation  
- Aggregation of sales, profit, and customer metrics  
- Query-based analysis for performance evaluation  

###  Python
Used for exploratory data analysis and visualization:
- **Pandas** – data manipulation and analysis  
- **NumPy** – numerical computations  
- **Matplotlib** – basic plotting and trend analysis  
- **Seaborn** – advanced statistical visualizations  

###  Power BI
- Interactive dashboards  
- KPI tracking using DAX  
- Filters and slicers for dynamic analysis
